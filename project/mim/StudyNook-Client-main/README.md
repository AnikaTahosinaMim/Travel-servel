# 🏠 StudyNook – Room Booking Platform

🌐 Live Site: https://notebook-booking.vercel.app

---

## 📖 About the Project
StudyNook is a modern room booking platform where users can find, add, and book rooms for study or meetings. It includes authentication, room management, and booking features.

---

## 🚀 Key Features

- 🔐 Google Login & Secure Authentication system
- 🏠 Users can add, edit, and delete rooms
- 📅 Real-time room booking system with time conflict checking
- 🔍 Search and filter rooms by name, amenities, and availability
- 📊 Dashboard for managing personal listings and bookings
- ⚡ Fast and responsive UI built with Next.js

---

## 🛠️ Tech Stack

- Next.js
- React
- Node.js
- Express.js
- MongoDB
- JWT Authentication
